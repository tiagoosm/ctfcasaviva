# 🚩 CTF Casaviva

A Capture The Flag (CTF) challenge developed for educational purposes at Inatel Cas@viva, designed to introduce students to cybersecurity through practical problem-solving.

This project focuses on logic, observation, and basic cryptography, offering interactive challenges where users analyze clues and submit flags to progress.

---

## 🎯 Objective

The goal of this project is to provide a hands-on introduction to cybersecurity concepts in an accessible and engaging way, encouraging students to think critically and explore problem-solving strategies.

---

## 👥 Target Audience

- Students from Inatel Cas@viva  
- Beginners interested in cybersecurity  
- Anyone looking to practice logical reasoning and basic cryptography  

---

## 🧠 How It Works

The platform presents a series of interactive challenges where users must:

- Analyze clues  
- Identify hidden patterns  
- Solve logic-based problems  
- Submit the correct flag to advance  

Each level is designed to stimulate curiosity and reinforce foundational cybersecurity skills.

## ⚙️ Getting Started (Local Setup)

```bash
# Clone the repository
git clone https://github.com/tiagoosm/ctfcasaviva.git

# Enter the project folder
cd ctfcasaviva

# Install dependencies
npm install

# Run the project
npm run dev
```
